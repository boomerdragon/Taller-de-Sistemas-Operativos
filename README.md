# 🖥️ Taller de Sistemas Operativos
### TecNM Campus Querétaro | Ingeniería en Sistemas Computacionales

Este repositorio contiene el material, las guías de configuración y la documentación técnica de la materia **Taller de Sistemas Operativos**. El enfoque del curso es 100% práctico, basado en la implementación de un **Laboratorio Híbrido** de servidores.

---

## 📅 Estructura del Curso

El semestre se divide en cuatro fases críticas para el dominio de la administración de servidores:

| Fase | Temática | Enfoque Principal |
| :--- | :--- | :--- |
| **I** | **Introducción y Virtualización** | Configuración de hipervisores y creación de entornos aislados. |
| **II** | **Sistemas Propietarios** | Administración avanzada de **Windows Server Core** (CLI). |
| **III** | **Software Libre** | Instalación y endurecimiento de **Linux Server** (Debian/Ubuntu). |
| **IV** | **Interoperabilidad y Proyecto** | Integración de servicios multiplataforma y Proyecto Integrador. |

---

## 🛠️ Tecnologías Utilizadas
* **Hipervisores:**
  * VirtualBox
  * VMware
* **Sistemas Operativos:**
  * Windows Server (Core Edition).
  * Distribuciones Linux (Debian, Ubuntu Server).
* **Servicios:**
  * Active Directory
  * DNS
  * DHCP
  * Web Servers (IIS/Apache/Nginx)
  * SSH
  * SAMBA.
* **Automatización:**
  * PowerShell
  * Bash Scripting.

---

## 📂 Organización del Repositorio

* `/Unidad_1_Virtualizacion`: Guías de instalación y configuración de red.
* `/Unidad_2_Windows_Server`: Scripts de PowerShell y bitácoras de Server Core.
* `/Unidad_3_Linux_Server`: Comandos de administración, gestión de paquetes y seguridad.
* `/Unidad_4_Interoperabilidad`: Guías para conectar servicios entre Windows y Linux.
* `/Proyecto_Integrador`: Documentación final del servidor híbrido funcional.
